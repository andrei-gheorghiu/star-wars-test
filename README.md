# star-wars

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
To view the production build locally:
```
npm i -g http-server
http-server dist
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
